# Вспомогательный скрипт для работы с файлами сэмплера

### Объединение отдельных паттернов в полноценный мультитрек
Использовать через терминал: 
```
waver comb "путь к папке с паттернами"
```

Пример для Windows:
```
.\waver comb "E:\PROJECT_07\"
```

Пример для Mac:
```
./waver comb "/FLASH/PROJECT_08"
```

Итоговые файлы будут лежать рядом с waver в папке out
